# Requirements Archive: v1.0 — Data + Classical Foundation

**Archived:** 2026-06-19
**Milestone scope:** Phases 1–2 (offline data + classical baselines).
**Core Value:** A user pastes text or a URL and instantly gets a trustworthy real/fake/malicious verdict with an understandable explanation and confidence score.

> **Important — partial v1.** REQUIREMENTS.md framed "v1" as the full 29-requirement product across Phases 1–7. Milestone **v1.0** ships only the **7 requirements in Phases 1–2**. The other **22 requirements are NOT shipped** — they are carried forward to subsequent milestones (tracked in PROJECT.md → Active). They are listed below as **DEFERRED**, deliberately *not* marked complete.

## Shipped in v1.0 (7/29)

### Data Foundation — Phase 1

- [x] **DATA-01** — Download scripts fetch source datasets (BanFakeNews/2.0, ISOT, LIAR, SMS Spam, phishing); raw not committed. — **validated**
- [x] **DATA-02** — Unified corpus with explicit real/fake/malicious label mapping + per-sample provenance. — **validated**
- [x] **DATA-03** — Source/publisher leakage detected + stripped (Reuters dateline, source-disjoint splits); leakage probe gate PASSED. — **validated** (caught a real first-run leak, fixed, re-ran clean)
- [x] **DATA-04** — Class imbalance handled (class-weighting at train time); leak-free source-disjoint 70/15/15 splits. — **validated** (natural distribution stored per D-03/D-04)
- [x] **DATA-05** — Shared `preprocess()` (incl. `csebuetnlp/normalizer`) used identically at train + inference for Bangla/English/code-mixed. — **validated**

### Classification & Models — Phase 2

- [x] **CLS-01** — Classical baselines: TF-IDF + Logistic Regression, Naive Bayes, Random Forest, trained + serialized. — **validated**
- [x] **CLS-03** — Compare all models on macro-F1 + per-class precision/recall + confusion (not accuracy); select best. — **validated** (best = LogisticRegression, test macro-F1 0.9140)
