# Milestone v1.0: Data + Classical Foundation

**Status:** ✅ SHIPPED 2026-06-19
**Phases:** 1-2
**Total Plans:** 10 (Phase 1: 7, Phase 2: 3)

## Overview

The offline foundation of the unified fake-news / malicious-content detector: a single de-leaked, documented, 3-class corpus (real / fake / malicious) with a shared train=inference preprocessing path, and trained + serialized classical TF-IDF baselines that lock the project's metric discipline (macro-F1 headline, leakage gates) before any GPU spend on the transformer.

> **Scope note:** This milestone covers **2 of the 7 planned phases** and **7 of the 29 v1 product requirements** (DATA-01..05, CLS-01, CLS-03). The remaining 22 requirements (transformer, malicious detection, signal modules, verification, fusion, explainability, Streamlit UI) are mapped to Phases 3–7, deferred to subsequent milestones. Completing this milestone ships the data+classical foundation, **not** the full v1 product.

## Phases

### Phase 1: Data Foundation

**Goal**: A single, documented, de-leaked, class-balanced 3-class corpus (real/fake/malicious) with grouped train/validation/test splits, plus one shared preprocessing function used identically offline and online.
**Depends on**: Nothing (first phase)
**Requirements**: DATA-01, DATA-02, DATA-03, DATA-04, DATA-05
**Plans**: 7 plans (6 waves)

- [x] 01-01: pytest scaffold + test stubs + shared preprocess() (DATA-05)
- [x] 01-02: dataset acquisition scripts → gitignored data/raw/ (DATA-01)
- [x] 01-03: label mapping (LIAR collapse) + provenance schema/Parquet (DATA-02)
- [x] 01-04: boilerplate/source-leakage stripping + language tagging (DATA-03)
- [x] 01-05: dedup (exact+fuzzy) + source-disjoint grouped 70/15/15 splits (DATA-04)
- [x] 01-06: build_corpus orchestrator → Parquet + class-distribution report (DATA-02/04)
- [x] 01-07: leakage probe gate (title/sentence/source-stripped) + report (DATA-03)

**Details:**
137,169-row corpus from 5 sources (BanFakeNews/2.0, ISOT, LIAR, SMS Spam, phishing) — train 103,801 / val 18,360 / test 15,008; real 70,412 / fake 33,308 / malicious 33,449; en 82,340 / bn 54,720 / code-mixed 107; 37.0% dedup removal (exact-SHA1 + MinHashLSH). Source-disjoint splits via GroupShuffleSplit + `assert_disjoint`. Leakage gate **closed after a real first-run FAIL**: bare in-body `Reuters` self-references survived the parenthesized-dateline strip; fixed with a third strip pass (bare `Reuters` → "the news agency", ISOT-only), corpus rebuilt deterministically, re-run PASSED (source_stripped macro-F1 0.9087 < 0.95, no surviving outlet/year tell). Shared `preprocess()` = `csebuetnlp/normalizer` + whitespace collapse only (lossless D-08), single importable entry point for train + inference.

### Phase 2: Classical Baselines + Metric Discipline

**Goal**: Trained, serialized TF-IDF classical baselines that validate the data pipeline and artifact-persistence path on CPU, with macro-F1 / per-class / confusion-matrix reporting established as the project's metric standard.
**Depends on**: Phase 1
**Requirements**: CLS-01, CLS-03
**Plans**: 3 plans (2 waves)

- [x] 02-01: hybrid word+char TF-IDF vectorizer (D-02) + metric-discipline module (CLS-01, CLS-03)
- [x] 02-02: SC-3 leakage re-check reusing leakage_probe leak-tells across LR/NB/RF + ≥0.98 flag (CLS-03)
- [x] 02-03: train/select/serialize orchestrator + models/ joblib artifacts + comparison report (CLS-01, CLS-03)

**Details:**
Hybrid TF-IDF = FeatureUnion(word(1,2) + char_wb(3,5)), min_df=2, max_features=50000, fit on TRAIN only (leak-safe). Models = LogisticRegression / ComplementNB / RandomForest with `class_weight='balanced'` + ComplementNB (D-01: no oversampling). Selection by VAL macro-F1 with a minority-recall guard (collapsed fake/malicious recall disqualified). **Best = LogisticRegression, test macro-F1 0.9140** (RF 0.9064, NB 0.6080); every model's fake + malicious recall > 0. SC-3 re-check gate fired correctly on ComplementNB (year tells 2016/2018 in top features → investigate), while the selected logreg is leak-clean (<0.98, no tells). Metric discipline locked: macro-F1 headline (never accuracy), per-class precision/recall, label-ordered confusion, per-language macro-F1, minority-recall guard.

## Milestone Summary

**Decimal Phases:** None.

**Key Decisions:**

- 3-class corpus assembled from incompatible sources via explicit per-source label mapping + retained provenance (gates all training).
- Offline→online boundary via serialized `models/` artifacts; classical **before** transformer to de-risk the data + artifact path before GPU spend.
- macro-F1 + per-class + confusion is the headline metric; ≥98% accuracy treated as suspected leakage (SC-3 gate).
- Natural class distribution stored + reported (D-03/D-04); balancing applied at train time via `class_weight`/ComplementNB, not resampling.
- Two-stage classifier (malicious gate → real/fake) decided here, implemented in Phase 3.
- BanFakeNews-2.0 4-class labels collapsed 3→real, 0/1/2→fake, documented with a report caveat (D-04).

**Issues Resolved:**

- Real source-leakage caught and fixed (bare-Reuters dateline) — leakage gate genuinely closed, not rubber-stamped.
